# 新概念英语练习王

一个为8岁儿童设计的新概念英语第一册学习程序，涵盖听说读写全方位学习。

## 功能特点

- 📚 **完整课程**：新概念英语第一册144课完整内容
- 🎤 **口语练习**：单词跟读、句子朗读、课文背诵
- 👂 **听力训练**：听写单词、听句答题、听文填空
- 📖 **阅读理解**：课文阅读、理解题目、拓展阅读
- ✍️ **写作练习**：看图写话、仿写句子、日记写作
- 🎮 **游戏化学习**：积分系统、成就徽章、进度可视化
- 🔊 **语音功能**：TTS语音合成，支持中英文发音
- 🎨 **儿童友好界面**：卡通风格、鲜艳色彩、动画效果

## 技术栈

- **应用框架**：Electron
- **前端技术**：React + TypeScript
- **UI设计**：自定义儿童主题组件
- **语音服务**：Web Speech API
- **数据存储**：SQLite
- **打包工具**：electron-builder

## 安装与运行

### 环境要求

- Node.js 16+
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

这将同时启动：
- Webpack开发服务器（端口3000）
- Electron应用

### 构建应用

```bash
# 构建生产版本
npm run build

# 打包Windows可执行文件
npm run dist
```

## 使用说明

1. **启动应用**：双击运行或在开发模式下启动
2. **选择课程**：在课程列表中选择要学习的课程
3. **学习内容**：
   - 单词学习：查看单词、音标、释义，点击发音按钮听发音
   - 课文学习：阅读课文，点击朗读按钮听全文
   - 练习：完成听力、口语、拼写等练习
4. **查看进度**：在学习进度页面查看学习统计

## 注意事项

1. **数据准确性**：课程数据需要仔细核对，确保单词、句子、课文准确无误
2. **语音质量**：TTS发音可能不够标准，建议关键内容使用预录音频
3. **儿童友好**：界面设计要简洁直观，操作要简单易懂
4. **性能优化**：注意内存使用，避免卡顿
