# 新概念英语第一册学习程序 - 实施计划

## 一、项目概述

### 1.1 项目目标
为8岁儿童开发一个新概念英语第一册的学习程序，涵盖听说读写全方位学习，包含单词、句子、课文、语法、时态、词汇结构等内容，兼顾趣味性和教育性。

### 1.2 核心需求
- **学习内容**：新概念英语第一册完整144课
- **学习维度**：听说读写全方位
- **技术要求**：Electron桌面应用，可编译为.exe本地运行
- **准确性**：单词、句子、课文内容准确
- **趣味性**：声音、图像、动画俱全，儿童友好界面

### 1.3 技术选型
- **应用框架**：Electron（桌面应用）
- **前端技术**：React + TypeScript
- **UI框架**：Ant Design / 自定义儿童主题组件
- **语音方案**：Web Speech API (TTS语音合成)
- **数据存储**：SQLite（本地数据库）
- **打包工具**：electron-builder

## 三、核心功能模块设计

### 3.1 课程管理模块
**功能描述**：管理144课的课程内容、分类、索引

**数据结构**：
```typescript
interface Lesson {
  id: number;                    // 课程编号 1-144
  title: string;                 // 课程标题
  type: 'new' | 'review';        // 新课/复习课
  grammar: string;               // 语法重点
  tense?: string;                // 时态
  vocabulary: Vocabulary[];      // 单词列表
  sentences: Sentence[];         // 句子列表
  text: TextContent;             // 课文内容
  exercises: Exercise[];         // 练习题
  illustration?: string;         // 插图路径
}

interface Vocabulary {
  word: string;                  // 英文单词
  phonetic: string;              // 音标
  meaning: string;               // 中文释义
  examples: string[];            // 例句
  image?: string;                // 配图
}

interface Sentence {
  english: string;               // 英文句子
  chinese: string;               // 中文翻译
  grammarPoints: string[];       // 语法点
}

interface TextContent {
  title: string;                 // 课文标题
  paragraphs: Paragraph[];       // 段落
  translation: string[];         // 中文翻译
}

interface Exercise {
  type: 'listening' | 'speaking' | 'reading' | 'writing';
  content: any;                  // 练习内容
  answer: any;                   // 答案
  score: number;                 // 分值
}
```

## 四、界面设计规范

### 4.1 设计原则
- **儿童友好**：大按钮、清晰图标、鲜艳色彩
- **简洁直观**：操作简单，流程清晰
- **趣味性强**：动画效果、音效反馈、角色互动
- **响应式布局**：适配不同屏幕尺寸

### 4.2 色彩方案
- 主色调：温暖的橙色、蓝色
- 辅助色：绿色（正确）、红色（错误）、黄色（提示）
- 背景：浅色系，护眼舒适

### 4.3 字体规范
- 标题：大号字体，加粗
- 正文：中等字号，清晰易读
- 英文：专门字体，便于识别

### 4.4 交互设计
- 按钮点击：动画反馈 + 音效
- 正确答案：绿色勾 + 欢快音效 + 星星动画
- 错误答案：红色叉 + 鼓励提示 + 再试一次
- 完成任务：庆祝动画 + 积分增加

### 6.2 TTS语音服务
```typescript
// src/services/tts.ts
class TTSService {
  private synth: SpeechSynthesis;

  constructor() {
    this.synth = window.speechSynthesis;
  }

  speak(text: string, lang: string = 'en-US', rate: number = 1) {
    const utterance = new SpeechSynthesisUtterance(text);
    utterance.lang = lang;
    utterance.rate = rate;
    this.synth.speak(utterance);
  }

  stop() {
    this.synth.cancel();
  }
}
```

### 6.3 数据库设计
```sql
-- 课程表
CREATE TABLE lessons (
  id INTEGER PRIMARY KEY,
  title TEXT NOT NULL,
  type TEXT CHECK(type IN ('new', 'review')),
  grammar TEXT,
  tense TEXT,
  content TEXT,  -- JSON格式存储完整内容
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 学习进度表
CREATE TABLE progress (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  lesson_id INTEGER,
  user_id INTEGER,
  completed BOOLEAN DEFAULT FALSE,
  score INTEGER,
  time_spent INTEGER,  -- 秒
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (lesson_id) REFERENCES lessons(id)
);

-- 用户表
CREATE TABLE users (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  avatar TEXT,
  total_score INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
