## 数据录入指南

### 课程数据格式

每课数据包含：
```typescript
{
  id: number,              // 课程编号 1-144
  title: string,           // 课程标题
  type: 'new' | 'review',  // 新课/复习课
  grammar: string,         // 语法重点
  tense?: string,          // 时态
  content: {
    vocabulary: [],        // 单词列表
    sentences: [],         // 句子列表
    text: {},              // 课文内容
    exercises: []          // 练习题
  }
}
```

### 数据文件位置

- 示例数据：`src/data/lessons/sampleLessons.ts`
- 完整数据：需要创建 `src/data/lessons/lesson1.ts` 到 `lesson144.ts`

### 数据录入步骤

1. 参考新概念英语教材
2. 按照数据格式录入每课内容
3. 确保单词、音标、释义准确
4. 确保句子翻译准确
5. 设计配套练习题

## 常见问题

### Q: npm install 失败？
A: 检查Node.js版本（需要16+），尝试使用国内镜像源

### Q: 启动失败？
A: 检查端口3000是否被占用，查看控制台错误信息

### Q: 语音不工作？
A: 确保浏览器支持Web Speech API，检查系统语音设置

### Q: 数据库错误？
A: 删除应用数据目录下的数据库文件，重新初始化

1. **小步快跑**：先实现核心功能，再逐步完善
2. **测试驱动**：每完成一个功能就测试
3. **数据优先**：确保课程数据准确完整
4. **用户体验**：站在儿童角度设计界面和交互
5. **性能优化**：注意内存使用和响应速度
