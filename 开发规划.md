### Phase 1: 跑通基础（让程序真正能用）

这是最关键的一步，完成后程序才从"能看"变成"能用"。

**1.1 修复数据库初始化**
- 在 `electron/main.ts` 的 `app.whenReady()` 中调用 `initDatabase()`
- 添加课程数据种子逻辑：启动时将 `completeLessons.ts` 的数据写入 SQLite
- 影响文件：`electron/main.ts`

**1.2 抽取 TTS 语音服务**
- 创建 `src/services/tts.ts`，把 LessonDetail 里的 `speakWord`、`speakText`、`getBestVoice` 提取为独立服务类
- 支持语速调节（慢速 0.6 / 正常 0.85 / 快速 1.1）
- 支持中英文切换
- 影响文件：新建 `src/services/tts.ts`，修改 `LessonDetail/index.tsx`

**1.3 为全部 144 课生成练习题**
- 编写自动生成脚本，从每课的单词、句子、课文数据中自动派生出 5 种练习题
- 每课生成 6-8 道题，覆盖以下类型：
  - **听力选择题**（listening）：播放单词/句子 → 从 4 个选项选正确答案
  - **口语跟读**（speaking）：显示句子 → 录音跟读 → 评分
  - **单词拼写**（spelling）：显示音标和中文 → 输入英文拼写
  - **阅读填空**（writing）：句子挖空 → 从选项选正确单词
  - **阅读理解**（reading）：基于课文段落 → 选择题
- 影响文件：`src/data/lessons/completeLessons.ts`

### Phase 2: 进度系统（让学习成果可见）

**2.1 接通进度保存**
- 练习完成后调用 `electronAPI.saveProgress()` 保存到 SQLite
- 记录：课程ID、得分、用时、是否完成
- 影响文件：`LessonDetail/index.tsx`

**2.2 首页统计接入真实数据**
- 首页的"总积分"、"已学课程"、"获得徽章"从数据库查询真实数据
- 影响文件：`Home/index.tsx`

**2.3 创建学习进度页面**
- 新建 `/progress` 路由和 Progress 页面
- 展示内容：
  - 课程完成进度条（已完成/总数）
  - 每课得分一览（网格视图，颜色区分完成状态）
  - 学习统计（总用时、平均正确率、连续学习天数）
- 影响文件：新建 `src/pages/Progress/`，修改 `App.tsx` 添加路由

### Phase 3: 代码重构（让代码可维护）

**3.1 拆分 LessonDetail 组件**

将 534 行的巨型组件拆分为：
- `src/components/lesson/VocabularyCard.tsx` — 单词卡片（含发音按钮）
- `src/components/lesson/TextDisplay.tsx` — 课文展示（双语对照 + 朗读）
- `src/components/practice/PracticeMenu.tsx` — 练习类型选择菜单
- `src/components/practice/PracticeEngine.tsx` — 练习引擎（统一处理 5 种题型）
- `src/components/practice/PracticeResult.tsx` — 练习结果展示

**3.2 创建课程服务层**
- `src/services/lesson.ts` — 课程数据加载（优先数据库，回退本地数据）
- `src/services/progress.ts` — 进度查询和保存的封装
- `src/services/achievement.ts` — 成就检测逻辑

### Phase 4: 语音识别评分（让口语练习真正有效）

**4.1 接入 Web Speech Recognition API**
- 创建 `src/services/speechRecognition.ts`
- 将口语练习的"随机分数"替换为真实语音识别
- 实现逻辑：录音 → 语音转文字 → 与目标文本比对 → 基于匹配度评分
- 评分维度：准确度（单词匹配率）、完整度（是否说完）
- 支持回退：浏览器不支持时降级为模拟评分 + 提示

### Phase 5: 游戏化激励系统（让孩子爱上学习）

**5.1 积分规则**
- 练习完成：按得分计入总分
- 课程完成：首次完成某课所有练习 → 额外奖励 50 分
- 连续学习：连续 3 天学习 → 奖励 30 分

**5.2 成就徽章**
- 学习之星：完成第 1 课
- 单词达人：累计拼写正确 50 个单词
- 口语高手：口语练习累计 10 次 80 分以上
- 听力专家：听力练习正确率达 90%
- 学霸：完成全部 144 课
- 坚持不懈：连续学习 7 天

**5.3 首页展示**
- 接入真实积分和徽章数据
- 添加进度环形图
- 添加今日学习提醒
